CSS (Cascading Style Sheets) — це мова стилів, яка використовується для оформлення веб-сторінок. Вона дозволяє змінювати кольори, шрифти, розташування елементів та їх розміри, щоб зробити сторінку привабливою та зручною для користувачів.
